Warren's Abstract Machine - Implementation by Stefan Buettcher - Ported to PHP by Trismegiste

developed: December 2001 until February 2002 ported: July 2012

Compiler contains the base class Compiler, which both QueryCompiler and PrologCompiler have been derived from. Additionally, it contains a class KeyValue, which is used for implementing mappings from Prolog variable names ("X", "A13", "Name", ...) to WAM variable names ("Y1", "Y2", ...).
